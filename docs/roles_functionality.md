# Функционал ролей — Электронный школьный журнал

## Роли в системе

| Роль | Описание |
|------|----------|
| `admin` | Полный доступ ко всему |
| `vice_principal` | Завуч — управление классами и расписанием, только свой профиль |
| `teacher` | Учитель — свои уроки, оценки и посещаемость своих классов |
| `student` | Ученик — просмотр своих данных |
| `parent` | Родитель — просмотр данных своих детей |

---

## Администратор (`admin`)

### Аккаунт и профиль
- Просмотр своего профиля (`GET /users/me`)
- Редактирование любого пользователя: email, пароль, имя, роль, статус активности
- Создание пользователей любой роли (включая других администраторов)
- Удаление любого пользователя

### Пользователи
- Просмотр полного списка всех пользователей
- Просмотр любого пользователя по ID
- Создание пользователей всех ролей
- Редактирование любого пользователя (все поля, включая `role` и `is_active`)
- Удаление любого пользователя
- Создание/редактирование профиля учителя (привязка предмета)
- Создание/редактирование профиля ученика (дата рождения, номер личного дела)
- Привязка ребёнка к родителю
- Отвязка ребёнка от родителя

### Предметы
- Просмотр списка предметов
- Просмотр предмета по ID
- Создание предмета
- Редактирование предмета
- Удаление предмета (если не используется в уроках/профилях учителей)

### Классы
- Просмотр списка всех классов
- Просмотр класса по ID
- Создание класса
- Редактирование класса (название, год, завуч)
- **Удаление класса** (только администратор)
- Просмотр списка учеников класса
- Добавление ученика в класс
- Удаление ученика из класса
- Просмотр списка учителей класса
- Прикрепление учителя к классу
- Открепление учителя от класса

### Уроки
- Просмотр всех уроков (без фильтрации)
- Просмотр урока по ID
- Создание урока (любой класс, любой учитель)
- Редактирование урока (тема, дата, класс, учитель, предмет)
- Удаление урока

### Оценки
- Просмотр всех оценок (без фильтрации)
- Просмотр оценки по ID
- Выставление оценки (любой ученик, любой урок)
- Редактирование оценки (значение, комментарий)
- Удаление оценки

### Посещаемость
- Просмотр всех записей посещаемости (без фильтрации)
- Просмотр записи по ID
- Добавление записи посещаемости
- Редактирование статуса посещаемости
- Удаление записи посещаемости

### Уведомления
- Просмотр своих уведомлений (все / только непрочитанные)
- Просмотр любого уведомления по ID (не только своего)
- Отправка уведомления любому пользователю
- Отметка своего уведомления как прочитанного
- Удаление любого уведомления (в том числе чужого)

## Завуч (`vice_principal`)

### Аккаунт и профиль
- Просмотр своего профиля (`GET /users/me`)
- Редактирование **только своего** профиля (имя, email, пароль)
- **Не может** изменить свою роль или статус активности
- **Не может** редактировать профили других пользователей

### Пользователи
- Просмотр полного списка всех пользователей
- Просмотр любого пользователя по ID
- Создание пользователей всех ролей **кроме** `admin`
- Создание/редактирование профиля учителя
- Создание/редактирование профиля ученика
- Привязка/отвязка ребёнка к родителю
- **Не может** удалять пользователей
- **Не может** изменять `role` и `is_active` других пользователей

### Предметы
- Просмотр списка предметов
- Просмотр предмета по ID
- Создание предмета
- Редактирование предмета
- Удаление предмета (если не используется)

### Классы
- Просмотр списка всех классов
- Просмотр класса по ID
- Создание класса
- Редактирование класса
- **Не может** удалять классы
- Добавление ученика в класс
- Удаление ученика из класса
- Прикрепление учителя к классу
- Открепление учителя от класса

### Уроки
- Просмотр всех уроков (без фильтрации)
- Просмотр урока по ID
- Создание урока
- Редактирование урока (все поля)
- Удаление урока

### Оценки
- Просмотр всех оценок (без фильтрации)
- Просмотр оценки по ID
- Выставление оценки
- Редактирование оценки
- Удаление оценки

### Посещаемость
- Просмотр всех записей посещаемости (без фильтрации)
- Просмотр записи по ID
- Добавление записи посещаемости
- Редактирование статуса посещаемости
- Удаление записи посещаемости

### Уведомления
- Просмотр **только своих** уведомлений
- Отправка уведомления любому пользователю
- Отметка своего уведомления как прочитанного
- Удаление **только своих** уведомлений

## Учитель (`teacher`)

### Аккаунт и профиль
- Просмотр своего профиля (`GET /users/me`)
- Редактирование **только своего** профиля

### Пользователи
- Просмотр списка пользователей — **только ученики и родители из своих классов**
- **Не может** создавать, редактировать или удалять пользователей

### Предметы
- Просмотр списка предметов
- Просмотр предмета по ID

### Классы
- Просмотр **только своих классов** (где прикреплён как учитель)
- Просмотр учеников и учителей своих классов
- **Не может** создавать, редактировать, удалять классы
- **Не может** добавлять/удалять учеников или учителей в классы

### Уроки
- Просмотр **только своих уроков** (где `teacher_id = свой ID`)
- Просмотр урока по ID (только своего)
- Редактирование **только темы** урока (дату изменить нельзя)
- **Не может** создавать уроки
- **Не может** удалять уроки

### Оценки
- Просмотр оценок учеников **только своих классов**
- Просмотр оценки по ID (только своих классов)
- Выставление оценки ученику **только своего класса**
- Редактирование оценки **только своих классов**
- Удаление оценки **только своих классов**

### Посещаемость
- Просмотр посещаемости учеников **только своих классов**
- Просмотр записи по ID (только своих классов)
- Добавление записи посещаемости **только своих классов**
- Редактирование статуса посещаемости **только своих классов**
- Удаление записи посещаемости **только своих классов**

### Уведомления
- Просмотр **только своих** уведомлений (все / непрочитанные)
- Отправка уведомления **любому** пользователю
- Отметка своего уведомления как прочитанного
- Удаление **только своих** уведомлений
